## Running locally
1. `git clone https://github.com/denipolis/coolor.git`
2. `cd coolor`
3. `pnpm`
4. `pnpm dev`

## Building
1. `git clone https://github.com/denipolis/coolor.git`
2. `cd coolor`
3. `pnpm`
4. `pnpm build`
